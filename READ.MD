This project is a simple book catalog website that displays books along with their cover, genre, and description. The goal of the site is to provide visitors with a clean and easy way to browse different books and learn basic information about each one. No user accounts, comments, or ratings are included â€” the site focuses only on presenting book details in a straightforward format.
The backend of this project is implemented with Express.js, while MongoDB is used as the primary database for storing book information.

--How to Install Packages and Run the Project?
---Backend Setup

1. cd server - navigate to the folder

2.npm install - install all required packages

3.npm start - start the backend server


---Frontend Setup

1.cd client - navigate to the folder

2.npm install - install all dependencies

3.npm run dev - run the development server
